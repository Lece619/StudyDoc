JSP(Java Server Page)
	HTML을 중심으로 자바와 같이 연동하여 사용하는 웹 언어이다.
	HTML 코드 안에 JAVA 코드를 작성할 수 있는 언어이다.(방식이다)

서버(Service + er ) 서비스를 제공해주는 것 =응답 
ex) 로그인 요청(사용자) -> 응답(service) 
	서버 : 사용자의 요청에 맞는 서비스를 제공해 주는 것
			
			요청(request)
	클라이언트     ------------------------>	서버 
			응답(response)
	서버 	      ------------------------>	클라이언트           

웹(Web)
네트워크에 따라 데이터가 이동을 하는데 예를들어 아이디와 비밀번호를 요청해서 데이터가 이동하는데
네트워크선이 복잡하게 지나가게된다.

	웹: 페이지 요청과 응답이 일어나는 장소. 인터넷에 연결된 컴퓨터들을 통해 
	    사람들이 정보를 공유할 수 있는 정보 공간.


클라이언트 -> 요청-> httpdemon ->>  	웹서버 
	<-------바로 응답	   <<---	[정적 or
			   	 동적] 판단-동적(연산이 필요)-->웹 컨테이너 
클라이언트		   <<---	정적	<-연산해서 정적으로 <---  .java (서블릿) 

Web Application Server= 웹서버+웹컨테이너  =was-톰켓 웹서버-아파치
